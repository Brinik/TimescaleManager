# TimescaleManager
Серверная часть сервиса загрузки csv файлов. Проект сделан как выполненное тестовое задание.

## Инструкция по запуску сервиса
Чтобы запустить сервис из Visual Studio, нужно
1. Скачать и установить Postgresql, если он еще не установлен;
2. Скопировать репозиторий в папку на компьютере;
3. Установите свои имя пользователя и пароль в файле TimescaleManager/appsettings.json в строке "ConnectionString,
либо создайте свой файл пользовательских секретов;
3. Зайти через консоль в корневую папку проекта и ввести команду в Packer Manager Console:
```bash
update-database
```
4. При возникновении ошибок попробуйте удалить папку "Infrastructure/Migrations" и выполните команды:
```bash
add-migration InitialCreate
update-database
```

## Комментарии
Описание методов:
**POST/File** - Загрузка файла csv в БД;
**POST/Result/GetFiltered** - Запрос результатов расчетов значений по фильтрам запроса 
(Если указать пустое значение имени файла - будет искать все имеющиеся. Остальные поля обязательны);
**GET/Value/LastTen** - получения списка последних 10 значений, отсортированных по начальному времени 
запуска Date по имени заданного файла.
